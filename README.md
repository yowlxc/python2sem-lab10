## ОТЧЁТ

1. ML-разработчик

   [Скринкаст с подробным и наглядным объяснением работы с ML в проекте](https://drive.google.com/file/d/1raTk6k2zryJYHShzzrYh1x2gf1r57AHT/view?usp=sharing)

   Сначала были подготовлены данные: удалены некорректные записи, обработаны категориальные признаки, нормализованы числовые данные и сохранён готовый датасет. Затем были обучены несколько моделей — Logistic Regression, Random Forest и Gradient Boosting. После сравнения по метрике ROC-AUC была выбрана лучшая модель и сохранена. Далее модель была дообучена на всём датасете и подготовлена для использования в проекте. Также была реализована оценка качества модели и возможность получать прогнозы для новых данных. В результате была создана модель, которая на основе информации о клиенте прогнозирует вероятность одобрения кредита.

2. Backend Developer

   [Скринкаст работы Backend](https://drive.google.com/file/d/1nnOTlKqu09cO-w4os5f5j5AL0fceJaX9/view?usp=sharing)

   В рамках проекта мной была реализована backend-часть ML-сервиса на FastAPI. Были разработаны API для загрузки модели, получения предсказаний по данным клиента и пакетной обработки CSV-файлов. Также была организована загрузка и использование обученной модели, реализована предобработка входных данных и обработка ошибок. Дополнительно была выполнена интеграция backend с пользовательским интерфейсом и проведено тестирование основных сценариев работы сервиса. В результате была создана серверная часть веб-приложения, обеспечивающая взаимодействие между моделью машинного обучения и пользователем.
